# Angular Directive Lessons

This track teaches directives as reusable behavior attached to existing DOM or template structure.

| Lesson                                  | Route                                                                 | What it teaches                                                         |
| --------------------------------------- | --------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| 1. Attribute Directives                 | `/angular-directive-lessons/lesson-01-attribute-directives`           | Add reusable behavior or styling to an existing element.                |
| 2. Structural Directives                | `/angular-directive-lessons/lesson-02-structural-directives`          | Add or remove template views with `TemplateRef` and `ViewContainerRef`. |
| 3. Host Binding / Host Listener         | `/angular-directive-lessons/lesson-03-host-binding-listener`          | Bind host classes/styles and react to host events.                      |
| 4. Directive Inputs                     | `/angular-directive-lessons/lesson-04-directive-inputs`               | Configure one directive instance with inputs.                           |
| 5. Directive Composition                | `/angular-directive-lessons/lesson-05-directive-composition`          | Reuse directive behavior through `hostDirectives`.                      |
| 6. Permission / Feature Flag Directives | `/angular-directive-lessons/lesson-06-permission-feature-flag`        | Encapsulate conditional UI access checks.                               |
| 7. Tooltip, Autofocus, Click Outside    | `/angular-directive-lessons/lesson-07-tooltip-autofocus-clickoutside` | Build practical DOM behavior directives.                                |
| 8. Directive vs Component               | `/angular-directive-lessons/lesson-08-directive-vs-component`         | Decide whether to attach behavior or own layout.                        |
