Route Params

Route params let one route pattern serve many records. Instead of creating separate routes for every project, Angular uses a placeholder like :projectId and reads the actual value from the current URL.

What To Remember

  1. :projectId names a dynamic URL segment.
  2. routerLink can build a URL by passing path segments as an array.
  3. snapshot.paramMap is a one-time read.
  4. paramMap is the safer choice when the same component can stay alive while the URL changes.
