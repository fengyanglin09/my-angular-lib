What To Remember

  1. Async validators return a Promise or Observable.
  2. Angular runs async validators after sync validators pass.
  3. The control status becomes PENDING while debounce and async validation run.
  4. Use a separate flag when the UI needs to know the backend request phase.
  5. Make the field readonly during the backend call, not during the debounce wait.
  6. Return null when valid, or an error object when invalid.
  7. Block submit while the form is pending so you do not save before validation finishes.
