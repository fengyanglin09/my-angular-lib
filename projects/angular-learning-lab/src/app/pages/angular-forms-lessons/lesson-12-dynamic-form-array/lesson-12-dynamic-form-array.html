What To Remember

  1. Use FormArray when the user can add or remove repeated form sections.
  2. Use a row factory so every new row gets the same controls and validators.
  3. The template loops over lines.controls and uses the row index as formGroupName.
  4. Guard against removing the final row when the app needs at least one editable section.
  5. Use a form stream when derived values, such as totals, must update after edits.
