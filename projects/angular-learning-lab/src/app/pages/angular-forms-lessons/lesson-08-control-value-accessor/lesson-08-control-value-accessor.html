What To Remember

  1. A custom form control must implement ControlValueAccessor.
  2. writeValue receives values from the parent form.
  3. registerOnChange gives the custom control a way to update the parent form.
  4. registerOnTouched lets the custom control mark itself touched, often from blur, click, or touch interaction.
  5. setDisabledState lets the parent form disable the custom control.
