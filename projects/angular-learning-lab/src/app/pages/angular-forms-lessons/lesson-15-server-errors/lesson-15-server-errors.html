What To Remember

  1. Client validation runs before the backend request.
  2. Backend validation can still reject values that look valid to the browser.
  3. Use setErrors to attach backend field errors to controls.
  4. Merge server errors with existing errors so you do not erase client validation state.
  5. Clear old server errors before retrying a backend save.
