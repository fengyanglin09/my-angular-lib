What To Remember

  1. Use a nested FormGroup when the form value contains an object.
  2. Use formGroupName to enter a child group in the template.
  3. Use a FormArray when users can add or remove repeated rows.
  4. Use formArrayName plus an index-based formGroupName for array rows.
  5. Save with getRawValue() when you want the full nested object.
