Value Vs Raw Value

Disabled controls are easy to overlook. profileForm.value can omit disabled child controls, while getRawValue() returns the full form object. This matters when disabled fields still belong in the backend payload.

What To Remember

  1. form.value is the enabled-control value shape.
  2. form.getRawValue() is the complete form value shape.
  3. Disabled controls can still be valid business data.
  4. Use getRawValue() when read-only disabled fields must be saved.
  5. This difference is easy to miss because both APIs often look the same until a child control is disabled.
