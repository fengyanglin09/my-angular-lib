Reactive Form Basics

Reactive forms put the form model in TypeScript. The template binds to that model, validators update form state, and submit reads a structured value when the form is valid.

What To Remember

  1. FormControl owns one field value and its validation state.
  2. FormGroup owns a named collection of controls.
  3. formControlName connects an input to a control inside the group.
  4. Use markAllAsTouched() on failed submit so validation messages become visible.
