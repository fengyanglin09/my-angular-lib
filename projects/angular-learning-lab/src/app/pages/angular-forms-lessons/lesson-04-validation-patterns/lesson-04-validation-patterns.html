What To Remember

  1. Use control validators when one field can be checked by itself.
  2. Use group validators when a rule needs more than one control.
  3. A validator returns null when valid.
  4. A validator returns an error object, such as {{ '{' }} passwordMismatch: true {{ '}' }}, when invalid.
  5. Show errors after touch or submit so the form does not yell before the user interacts.
