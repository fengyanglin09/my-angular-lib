What To Remember

  1. map changes every item into a new value.
  2. filter keeps items that match a predicate.
  3. reject removes items that match a predicate.
  4. find returns the first match, not an array.
  5. sortBy sorts ascending; combine it with reverse for descending order.
  6. groupBy changes an array into an object of grouped arrays, not counts.
  7. If you want counts, call .length on each grouped array afterward.
