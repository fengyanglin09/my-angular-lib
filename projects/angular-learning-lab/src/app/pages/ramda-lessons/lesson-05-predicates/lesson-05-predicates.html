What To Remember

  1. A predicate is just a function that returns true or false.
  2. propEq checks one property against one expected value.
  3. where checks several properties with several predicate functions.
  4. allPass requires every predicate to pass.
  5. anyPass requires at least one predicate to pass.
  6. complement inverts another predicate.
