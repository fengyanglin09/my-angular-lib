What To Remember

  1. RxJS operators work on stream emissions over time.
  2. Ramda helpers work on the plain value inside one emission.
  3. RxJS map changes each emitted value; Ramda map changes each array item.
  4. Keep Ramda transformation logic in named functions when the RxJS pipeline starts getting busy.
  5. Plain TypeScript is still a good choice when the transformation is short and obvious.
