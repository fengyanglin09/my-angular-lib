What To Remember

  1. pick keeps named fields from one object.
  2. omit removes named fields from one object.
  3. project is like running pick for every object in an array.
  4. pluck collects one property from every object in an array.
  5. pick and omit can be configured first, then passed to map.
  6. applySpec builds a new object from several small field-producing functions.
  7. Use plain TypeScript when the object shape is small and obvious.
