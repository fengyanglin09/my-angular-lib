Immutable Object Updates

Ramda can return updated copies of objects without mutating the original object. This matters in Angular, NgRx, and Signals because many state patterns depend on replacing data with a new reference.

What To Remember

  1. assoc updates one top-level property and returns a new object.
  2. assocPath updates one nested path and returns a new object.
  3. evolve updates fields by running functions against the old values.
  4. None of these helpers mutate the object passed into them.
  5. Apply update replaces the current profile with the previewed copy.
