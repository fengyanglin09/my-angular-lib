Pipeable Operators

Operators are functions that sit inside pipe(). They receive values from the source Observable and can inspect, filter, or transform those values before the subscriber receives them.

map, filter, tap

What To Remember

  1. pipe() chains operators together.
  2. tap observes values but returns the same value onward.
  3. filter removes values that do not pass a condition.
  4. map transforms each value into a new value.
  5. The subscriber only receives values that survive the full pipeline.
