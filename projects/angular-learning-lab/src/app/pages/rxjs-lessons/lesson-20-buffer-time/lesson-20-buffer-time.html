Batching Events With bufferTime

The bufferTime operator collects values for a period of time, then emits them as an array. It is useful when a backend should receive batches instead of many tiny requests.

What To Remember

  1. bufferTime emits arrays of collected values.
  2. Empty arrays can be filtered out with filter((batch) => batch.length > 0).
  3. It is useful when many small events should become fewer backend requests.
  4. For count-based batches, RxJS also has bufferCount.
