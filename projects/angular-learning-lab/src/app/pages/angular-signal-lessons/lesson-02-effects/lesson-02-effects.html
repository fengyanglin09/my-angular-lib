Effects For Side Effects

computed is for derived values you display or reuse. effect is for work that should happen because signal values changed, such as logging, syncing preferences, or calling browser APIs.

Mental Model

computed
  returns a derived value
  should stay pure
  use it in template/code

effect
  runs because signals changed
  performs side effects
  does not return display state

What To Remember

  1. effect(...) runs once when created.
  2. Angular tracks every signal read inside the effect.
  3. When a tracked signal changes, the effect runs again.
  4. Use computed for values, and effect for side effects.
