linkedSignal

Mental Model

signal()
  independent writable state

computed()
  derived read-only state

linkedSignal()
  derived default
  writable by user
  resets when dependencies change

What To Remember

  1. linkedSignal creates writable state.
  2. It reads other signals in its computation, like computed.
  3. When those dependencies change, the linked signal resets from the computation.
  4. The user can still manually change it with set or update.
