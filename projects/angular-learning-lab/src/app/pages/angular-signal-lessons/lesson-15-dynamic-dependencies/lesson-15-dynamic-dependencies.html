How To Try It

  1. Stay in draft mode and change the draft fields. The preview effect reruns.
  2. Stay in draft mode and change published fields. The visible preview changes do not depend on them, so the preview effect does not rerun.
  3. Switch to published mode. Angular reruns the computed values and starts tracking the published fields.
  4. Now edit draft fields again. They are no longer active dependencies for the preview.
