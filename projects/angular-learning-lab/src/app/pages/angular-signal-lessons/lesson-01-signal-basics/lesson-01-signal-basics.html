Angular Signals Lesson 1

Signal Basics

Signals are Angular's local reactive state primitive. A writable signal stores a value, and a computed signal derives a value from other signals and updates automatically.

Mental Model

writable signal
  stores a value
  read with quantity()
  change with set/update

computed signal
  reads other signals
  caches the derived value
  recalculates when dependencies change

What To Remember

  1. signal(value) creates writable state.
  2. Read a signal by calling it like a function: quantity().
  3. set(...) replaces the value.
  4. update(...) calculates the next value from the current value.
  5. computed(...) is for derived values, not manually assigned values.
