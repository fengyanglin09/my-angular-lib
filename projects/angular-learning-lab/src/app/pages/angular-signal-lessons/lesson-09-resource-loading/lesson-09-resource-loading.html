What To Remember

  1. resource is intended for read/loading workflows.
  2. params is reactive; changing signals inside it triggers a new load.
  3. loader receives params, previous, and abortSignal.
  4. The loader can delegate backend work to a service and pass abortSignal along.
  5. Use status(), isLoading(), error(), and value() in the template.
