Defer Blocks

@defer lets Angular delay loading and rendering expensive UI until a trigger is met. It is useful for large charts, editors, maps, and secondary panels that do not need to be part of the first render.

Mental Model

initial render
  placeholder appears

trigger becomes true
  deferred dependency loads
  loading block may appear

after loaded
  component behaves normally
  inputs keep updating

What To Remember

  1. @defer delays loading and rendering a block until a trigger is satisfied.
  2. @placeholder renders before the deferred block starts.
  3. @loading renders while the deferred dependency is loading.
  4. Once loaded, the deferred component still updates normally when its inputs change.
