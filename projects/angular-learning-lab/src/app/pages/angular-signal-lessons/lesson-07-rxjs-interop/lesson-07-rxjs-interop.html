  return async state
  template reads signal

What To Remember

  1. toObservable(signal) turns signal changes into Observable emissions.
  2. Use RxJS operators when you need debounce, cancellation, retries, or stream composition.
  3. toSignal(observable, ...) gives the template signal-style state again.
  4. Provide an initialValue so the template has a value before the Observable emits.
