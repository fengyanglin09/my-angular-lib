What To Remember

  1. JSON Server is a lightweight local REST API, not something GitHub Pages can host.
  2. Angular still uses normal HttpClient calls.
  3. The backend URL changes from a static file path to http://localhost:3000/courses.
  4. POST, PATCH, and DELETE change mock-api/db.json while the local server is running.
  5. If the server is not running, Angular receives a connection error and should show a useful message.
