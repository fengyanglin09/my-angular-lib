What To Remember

  1. Add provideHttpClient() once in app configuration before injecting HttpClient.
  2. http.get<CourseDto[]>(url) tells TypeScript what shape you expect back.
  3. The generic type is a compile-time expectation; the backend response still needs to actually match it.
  4. HTTP Observables usually emit once and complete.
  5. Keep URLs and DTO types in an API service so components stay focused on UI state.
