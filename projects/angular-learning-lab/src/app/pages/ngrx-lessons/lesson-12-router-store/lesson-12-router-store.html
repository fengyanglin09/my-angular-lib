Router Store

Router Store copies Angular Router state into NgRx. Components can select the current URL, route params, and query params from Store just like any other state.

What Router Store Adds

  1. Angular navigation dispatches router actions.
  2. The router reducer stores the latest router snapshot.
  3. Router selectors read URL, params, query params, data, and fragments.
  4. Components can react to navigation through Store selectors.
