Effect Flow

  1. Router Store updates when the URL changes.
  2. The route effect selects projectId, view, and url from Store.
  3. distinctUntilChanged prevents duplicate loads for the same route state.
  4. The effect dispatches Load Project From Route.
  5. A second effect calls the backend and dispatches success or failure.
