Effect Flow

  1. The component dispatches Save Note For Selected Book with only note text.
  2. The effect pairs that action with selectSelectedBook from Store.
  3. If no book is selected, the effect dispatches failure.
  4. If a book is selected, the effect saves note plus book details.
