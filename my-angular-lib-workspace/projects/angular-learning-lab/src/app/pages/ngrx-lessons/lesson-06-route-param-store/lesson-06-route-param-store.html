New Idea In This Lesson

  1. The component uses ActivatedRoute in ngOnInit.
  2. The route id is passed into projectStore.loadProject(projectId).
  3. The Signal Store owns backend loading, saving, loading state, and errors.
  4. Changing links reuses the component while loading a different project id.
  5. This is useful when store initialization depends on URL params.
