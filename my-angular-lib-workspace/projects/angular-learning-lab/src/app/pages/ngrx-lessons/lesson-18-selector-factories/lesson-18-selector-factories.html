Selector Factories

A selector factory is a function that receives a parameter and returns a selector. Use it when the same selection logic needs different inputs, like book id or author name.

What To Remember

  1. selectBookById(3) creates a selector for one specific id.
  2. selectBooksByAuthor('Ada Weaver') creates a selector for one author.
  3. Create selector instances in TypeScript, not repeatedly in the template.
  4. Selector factories are useful for ids, categories, authors, tabs, or route params.
