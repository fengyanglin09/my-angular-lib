When To Use Each

  1. Use of when you already have values and want to emit them.
  2. Use from when converting an array, promise, or iterable.
  3. Use interval for repeated scheduled emissions.
  4. Use timer for delayed start or one delayed emission.
  5. Use take to limit streams that would otherwise keep running.
