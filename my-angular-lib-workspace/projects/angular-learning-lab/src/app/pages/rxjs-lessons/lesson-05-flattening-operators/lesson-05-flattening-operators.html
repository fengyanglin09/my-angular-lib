What To Remember

  1. The outer Observable emits clicks. Each click creates an inner backend request Observable.
  2. switchMap keeps only the latest inner Observable.
  3. concatMap queues inner Observables in order.
  4. mergeMap lets inner Observables run at the same time.
  5. exhaustMap ignores new outer values while busy.
