Angular Subscription Patterns

In Angular, subscribing is not only about receiving values. It is also about ownership: TypeScript can own the subscription, Angular can clean it up with takeUntilDestroyed, or the template can own it with the async pipe.

What To Remember

  1. Use the async pipe when the template only needs to display Observable values.
  2. Use takeUntilDestroyed when TypeScript needs the values and the subscription should end with the component.
  3. Manual unsubscribe() works, but you must remember to do it.
  4. HTTP Observables usually complete by themselves, but long-lived streams like intervals, route params, and form changes need cleanup.
