NgRx Lesson 1

Counter Store

This first lesson follows the smallest useful NgRx loop: dispatch an action, update state in a reducer, and read the result with a selector.

Live Store Value

{{ countLabel() }}

{{ count() }}

What Just Happened?

  1. The button calls a component method.
  2. The method dispatches an NgRx action.
  3. The reducer receives the action and returns new state.
  4. The selector reads the count from the store.
  5. The template updates from the selected signal.
