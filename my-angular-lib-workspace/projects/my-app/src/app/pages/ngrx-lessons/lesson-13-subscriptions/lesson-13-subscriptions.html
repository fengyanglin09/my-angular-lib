Safe Subscriptions

Most UI can use signals or the async pipe. When component TypeScript needs a manual subscription for a side effect, clean it up with takeUntilDestroyed.

When To Subscribe Manually

  1. Use signals or async pipe when the template only needs to display state.
  2. Use manual subscribe when the component must perform a side effect.
  3. Pipe through distinctUntilChanged when repeated identical values should be ignored.
  4. Pipe through takeUntilDestroyed so Angular unsubscribes when the component is destroyed.
