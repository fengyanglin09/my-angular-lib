Functional Effects

Functional effects are exported functions instead of class properties. They still listen to actions, call services, and emit actions, but they use inject() for dependencies.

functional: true

Class Effect vs Functional Effect

  1. Class effects are properties on an injectable class.
  2. Functional effects are exported constants created with createEffect.
  3. Functional effects use inject() to get Actions and services.
  4. Register functional effects by passing their exported object to provideEffects.
