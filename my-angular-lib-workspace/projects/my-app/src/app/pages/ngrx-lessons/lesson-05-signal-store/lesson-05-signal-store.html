New Idea In This Lesson

  1. signalStore creates an injectable store class for this page.
  2. withHooks loads queue data from the backend when the store initializes.
  3. withState tracks items, loading, saving, errors, and last saved time.
  4. withComputed derives filtered lists, counts, and summary text.
  5. withMethods patches state and saves modified lists back to the API.
  6. The component provides the store, so this backend state is scoped to this page instance.
