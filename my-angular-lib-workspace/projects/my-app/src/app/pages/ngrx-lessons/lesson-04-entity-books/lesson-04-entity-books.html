Entity For Collections

New Idea In This Lesson

  1. The component dispatches loadBooks with page and page size.
  2. The effect calls the paginated API and dispatches success or failure.
  3. setAll replaces Entity state with the current page.
  4. addOne, updateOne, and removeOne still handle local changes.
  5. The save effect reads the current page collection with a selector.
  6. The API returns page metadata: page, page size, and total records.
