New Idea In This Lesson

  1. rxMethod exposes a store method powered by an RxJS pipeline.
  2. debounceTime waits for typing to pause before calling the backend.
  3. distinctUntilChanged ignores repeated identical queries.
  4. switchMap cancels stale backend requests when a newer query arrives.
  5. The store patches loading, results, and error state for the template.
